# Referencia de Seis8s


La premisa de seis8s es el poder interactuar con colecciones de conocimiento musical representado a través de bloques interlazados de código. En seis8s una coleccion de conocimiento se representa a través de un estilo musical. E.g. cumbia.

Dicho estilo necesita de un "cuerpo" para volverse tangible/visible (ie. audible en este caso). Por ejemplo un piano.

```
cumbia piano
```
